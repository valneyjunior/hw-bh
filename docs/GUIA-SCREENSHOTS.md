# Guia de captura de screenshots — Manual BH Pulse

Tire cada print abaixo e salve em **`docs/img/`** com o **nome exato** indicado.
O `MANUAL.md` já referencia esses arquivos — assim que salvá-los, as imagens aparecem.

**Dica:** use `Win + Shift + S` (Ferramenta de Captura do Windows) para recortar só a área do conteúdo.

## Credenciais para acessar cada perfil

| Perfil | Como acessar |
|---|---|
| **Administrador** | Sua conta (`valney.junior@hostweb.cloud`) |
| **Atendimento** | `atendimento.teste@hostweb.cloud` / `Atend@123` |
| **Analista / Coordenador** | Use uma conta existente desses perfis, ou peça para eu criar contas de teste |

> Para ver telas de Analista e Coordenador com dados, faça login com uma conta desses
> perfis. Se preferir, eu crio `analista.teste` e `coord.teste` com senha conhecida.
